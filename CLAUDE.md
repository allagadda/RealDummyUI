# Real Estate App — Figma Project

## Project Structure
- `assets/` — images, icons, fonts
- `components/` — reusable UI components
- `screens/` — full page/screen designs
- `tokens/` — design tokens (colors, spacing, typography)

## Figma MCP
Connected via `figma-developer-mcp`. Use Figma tools to read and update designs in real time.

## Notes
- Replace `YOUR_FIGMA_PERSONAL_ACCESS_TOKEN` in `~/.claude/settings.json` with your actual Figma PAT before use.
